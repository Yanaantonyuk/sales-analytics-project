**E-commerce Sales Analytics Project**

End-to-end analytics project built as part of a junior data analyst portfolio.
Period: November 2020 – January 2021 (Black Friday → New Year)
Currency: GBP (£)

🛠️ Tech Stack

SQL (BigQuery) — data extraction and joins
Python (Pandas) — cleaning, feature engineering, EDA
Excel — dataset export for Tableau
Tableau — interactive dashboards
